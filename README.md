# ⏳ Chronos

Chronos is an AI-powered productivity and scheduling assistant designed to help users organize their time more effectively. By combining intelligent scheduling with interactive features, Chronos aims to make planning both productive and enjoyable.

---

## 🚀 About the Project

This project was developed as part of the **Vibe 2 Ship Hackathon**, where our goal was to build an intelligent planning assistant capable of generating personalized schedules and battle plans based on user input.

---

## ✨ Features

* 📅 Smart calendar and schedule management
* 🤖 AI-generated battle plans and productivity suggestions
* ⚡ Fast and intuitive user experience
* 🎮 Calendar Tetris mini-game to keep users engaged while waiting for AI responses

## ⚠️ Notes

### AI Battle Plan Generation

If a battle plan is not generated, it is most likely due to heavy traffic or temporary rate limiting from the AI API provider.

Please wait a few moments and try again.

### While You Wait...

Don't worry—you don't have to stare at a loading screen!

You can play our built-in mini-game, **Calendar Tetris**, while your request is being processed.
