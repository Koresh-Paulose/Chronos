# Chronos

Chronos is an AI-powered productivity assistant that helps users organize their schedules and generate actionable "battle plans" to tackle their day. Built with a focus on simplicity and responsiveness, Chronos combines intelligent planning with a fun user experience.

---

# About the Project

Chronos was developed as part of the **Vibe 2 Ship Hackathon**. The objective was to build an AI-powered application that helps users manage their time more effectively while providing an engaging experience during AI processing.

---

#  Features

* 📅 Intelligent schedule management
* 🤖 AI-generated battle plans based on user input
* ⚡ Fast and responsive interface
* 🎮 **Calendar Tetris** mini-game to entertain users while waiting for AI responses
* ☁️ Cloud-backed data storage and authentication

# Notes

## AI Battle Plan Generation

If a battle plan is not generated, it is most likely due to temporary API congestion or rate limiting from the AI service.

Please wait a few moments and try again.

## While You're Waiting...

Instead of waiting on a loading screen, you can enjoy our built-in **Calendar Tetris** mini-game until your battle plan is ready.
